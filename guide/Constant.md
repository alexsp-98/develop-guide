## 常量&变量
### 1. 直接赋值常量值，禁止声明新对象

直接赋值常量值，只是创建了一个对象引用，而这个对象引用指向常量值。
反例
```java
Long i = new Long(1L);
String s = new String("abc");
```
正例
```java
Long i = 1L;
String s = "abc";
```
